AI Conversation Worker (Node.js + RabbitMQ)
Este projeto consiste em um Worker Node.js robusto projetado para processar solicitações de Inteligência Artificial de forma assíncrona. Utilizando o RabbitMQ como message broker, o sistema desacopla o front-end do processamento pesado da IA, garantindo que o usuário nunca enfrente timeouts e que o sistema possa escalar horizontalmente.
